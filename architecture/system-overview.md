## System Architecture Overview

Omniflow follows a modern, cloud-native architecture.

### Core Components
- Frontend: Web UI (Builder.io prototype)
- Backend: Serverless APIs
- Data Layer: Centralized relational database
- AI Layer: Rule-based and ML-assisted logic
- Notification Layer: Email automation
- Audit Layer: Immutable logs

### Design Principles
- Scalability
- Security
- Transparency
- Responsible AI usage
